# TaskPoint Android

## Requisitos

- Android Studio con Android SDK instalado.
- Un emulador Android o dispositivo fisico.
- Backend de TaskPoint ejecutandose.

## Levantar la app

1. Abrir esta carpeta (`TaskPoint`) en Android Studio.
2. Esperar a que termine la sincronizacion de Gradle.
3. Iniciar el backend siguiendo [backend/README.md](backend/README.md).
4. Seleccionar un emulador o dispositivo.
5. Presionar **Run** en Android Studio.

El emulador usa `http://10.0.2.2:8080` para conectarse al backend que se ejecuta en la misma PC.
